## 📈 项目概览

### 技术栈

**后端**
- Cloudflare Workers (Serverless)
- Hono (Web Framework)
- D1 (SQLite Database)
- KV (Key-Value Store)

**管理后台**
- React 18
- Ant Design Pro
- Refine (Admin Framework)
- Vite

**移动应用**
- Flutter 3.x
- GetX (State Management)
- Dio (HTTP Client)
- MediaKit (Video Player)

### 核心功能

✅ 用户认证和授权  
✅ 动态布局系统  
✅ 视频播放和 DLNA 投屏  
✅ 短剧双模式播放  
✅ 搜索和筛选  
✅ 广告系统  
✅ 用户中心  
✅ 反馈和预约  
✅ 应用墙  
✅ 版本管理  

### 项目特色

🎨 **后端驱动的动态配置**  
通过管理后台可视化编辑 APP 布局，无需发版即可调整内容展示

⚡ **Serverless 架构**  
零运维成本，自动扩展，全球 CDN 加速

🎬 **短剧双模式**  
支持抖音式无限流和详情页两种观看模式

📱 **跨平台支持**  
一套代码，同时支持 Android 和 iOS

🔒 **安全可靠**  
JWT 认证、API Key 鉴权、输入验证、崩溃上报
